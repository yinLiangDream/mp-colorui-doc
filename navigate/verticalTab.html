# VerticalTab 参数

参数 说明 类型 可选值 默认值
height 高度 number - -
tabs 标签页数组 tabs[] - []
current 默认激活的标签页 string - -
backTop 点击标题回到最上方 boolean - false

# tabs

参数 说明 类型 可选值 默认值
id Tab id string - -
name Tab 名称 string - -
