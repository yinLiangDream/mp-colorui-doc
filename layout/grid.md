## 使用说明

```jsx
import ClGrid from 'mp-colorui'
```



## 一般用法

### 分割成几格

```jsx
<ClGrid col={4} ></ClGrid>
```



## 参数说明

### Grid 参数

| 参数 | 说明             | 类型   | 可选值                | 默认值     |
| ---- | ---------------- | ------ | --------------------- | ---------- |
| col  | *一行分割成几格* | Number | -                     | 3          |
| mode | *类型*           | String | *`square`*,*`normal`* | *`normal`* |
